# AI Job Market Analysis

## Overview

This project explores AI-related job postings to identify salary patterns, industry trends, employment types, and in-demand technical skills.

Using Python and exploratory data analysis techniques, the dataset was cleaned, transformed, and visualised to uncover insights into the current AI job market.

---

## Dataset

The dataset contains approximately 2,000 AI-related job postings and includes:

* Job titles
* Industry sectors
* Employment types
* Posting dates
* Required skills
* Salary ranges

The data provides a snapshot of hiring trends and compensation patterns within the AI industry.

---

## Objectives

* Clean and preprocess job posting data
* Transform salary ranges into numerical values
* Explore salary trends across industries and job roles
* Analyse employment type distributions
* Investigate relationships between skills and salary
* Generate insights relevant to job seekers and employers

## Key Findings

### Salary Trends

* Average salaries were relatively similar across industries.
* Automotive and E-commerce sectors showed slightly higher salary levels.
* Salary ranges varied significantly between job postings.

### Skills Analysis

Frequently requested skills included:

* Python
* Machine Learning
* Natural Language Processing (NLP)
* Cloud Technologies

These skills were commonly associated with higher-paying positions.

### Industry Insights

* AI job opportunities were distributed across multiple industries.
* No single industry dominated the dataset.
* Hiring demand appears widespread across sectors adopting AI technologies.

### Correlation Analysis

Correlation analysis revealed that no single feature strongly predicts salary. Compensation appears to depend on multiple combined factors such as skills, role type, and industry.
